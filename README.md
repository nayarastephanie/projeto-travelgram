# Travelgram

Perfil/portfólio de viagens estático desenvolvido com HTML5 e CSS3.

## 📋 Sobre

Site pessoal de portfólio de viagens da viajante Isabela Torres. Apresenta estatísticas de viagem (37 países, 240 fotos) e uma galeria de destinos visitados.

## 🛠️ Tecnologias

- HTML5
- CSS3 (modularizado)
- Google Fonts (Poppins)
- SVG + PNG

## 📁 Estrutura

```
projeto-travelgram/
├── index.htm
├── assets/
│   ├── icons/
│   └── images/
└── styles/
    ├── global.css
    ├── index.css
    ├── header.css
    ├── nav.css
    ├── main.css
    └── footer.css
```

## 🚀 Como usar

1. Clone o repositório
2. Abra `index.htm` no navegador
